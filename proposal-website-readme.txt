# Heartfelt Message Website

This is a special website designed to convey a heartfelt message to someone important. The site features an interactive proposal with dynamic elements and a celebratory modal.

## Features

- Welcome alert message
- Beautifully styled content using Bootstrap and custom CSS
- Interactive "Perhaps" button that moves when hovered
- Celebratory modal triggered by acceptance
- Responsive design for various screen sizes

## Setup

1. Clone this repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.1.3

## Preview
